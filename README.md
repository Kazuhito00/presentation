# presentation-2020
 presentation-2020はopencv2webslidesを用いたプレゼンテーション資料をまとめたリポジトリです。

# Requirement
 
* OpenCV 3.4.2
* Eel 0.11.0
* Tensorflow 1.14.0
 
# Usage
 
presentation_documentディレクトリの配下にプレゼンテーション資料が格納されています。

確認したい資料のディレクトリにて以下のPythonプログラムを実行してください。
 
```bash
python presentation.py
```
